# Coursework 2 – Clustering & Multimodal Analysis

## 📌 Overview
This project explores unsupervised learning and multimodal data analysis using Python and scikit-learn.

The main focus is on clustering techniques and analysing relationships between image features and text (captions).

---

## 🧠 Objectives
- Apply clustering algorithms (KMeans) to real-world data
- Analyse similarity between data points using feature representations
- Explore multimodal relationships between images and text
- Evaluate clustering quality and interpret results

---

## ⚙️ Methods
- **Clustering:** KMeans
- **Distance Metrics:** Euclidean distance (pairwise distances)
- **Feature Analysis:** Image and caption embeddings
- **Evaluation:** Cluster separation and distance comparison

---

## 🛠️ Tech Stack
- Python
- NumPy
- scikit-learn
- Jupyter Notebook

---

## 📊 Key Learning
- Understanding how clustering works without labels (unsupervised learning)
- Importance of feature representation in ML models
- How distance metrics affect clustering results
- Practical experience with real ML workflows

---
