# Coursework 1 – Regression & Data Analysis

## 📌 Overview
This project focuses on applying supervised learning techniques to analyse relationships within real-world data using Python and scikit-learn.

The primary objective is to build and evaluate regression models to understand how input features influence continuous outputs.

---

## 🧠 Objectives
- Perform data cleaning and preprocessing
- Explore relationships between variables
- Train regression models (Linear Regression)
- Evaluate model performance using appropriate metrics

---

## ⚙️ Methods
- **Model:** Linear Regression
- **Loss Functions:** Mean Squared Error (MSE), Mean Absolute Error (MAE)
- **Data Processing:** Feature selection and preparation
- **Evaluation:** Model prediction vs actual values

---

## 🛠️ Tech Stack
- Python
- NumPy
- pandas
- scikit-learn
- Jupyter Notebook

---

## 📊 Key Learning
- Understanding regression models for continuous prediction
- Importance of loss functions in model evaluation
- Impact of data quality on model performance
- Experience with end-to-end ML workflow
